Documentation

The goal is to build the server-side component of a movie web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

User Endpoints

Create a User

Endpoint: POST /users

Description: Creates a new user with a unique ID. The request body must include a name field.

Request Body:

{
    "name": "string"
}

Responses:

  • 201 Created: Returns the newly created user object.
  • 400 Bad Request: If the name field is missing.

Add a Favorite Movie to a User

Endpoint: POST /users/:id/:movieTitle

Description: Adds a favorite movie to the specified user by user ID.

Path Parameters:

  • id: User ID
  • movieTitle: Title of the movie to be added to the user's favorites

Responses:

  • 200 OK: Confirms that the movie has been added.
  • 400 Bad Request: If the user does not exist.

Update a User's Name

Endpoint: PUT /users/:id

Description: Updates the name of the specified user by user ID.

Path Parameters:

  • id: User ID

Request Body:

{
    "name": "string"
}

Responses:

  • 200 OK: Returns the updated user object.
  • 400 Bad Request: If the user does not exist.

Delete a Favorite Movie from a User

Endpoint: DELETE /users/:id/:movieTitle

Description: Removes a favorite movie from the specified user by user ID.

Path Parameters:

  • id: User ID
  • movieTitle: Title of the movie to be removed from the user's favorites

Responses:

  • 200 OK: Confirms that the movie has been removed.
  • 400 Bad Request: If the user does not exist.

Delete a User

Endpoint: DELETE /users/:id

Description: Deletes the specified user by user ID.

Path Parameters:

  • id: User ID

Responses:

  • 200 OK: Confirms that the user has been removed.
  • 400 Bad Request: If the user does not exist.

Movie Endpoints

Get All Movies

Endpoint: GET /movies

Description: Retrieves a list of all top movies.

Responses:

  • 200 OK: Returns an array of movie objects.

Get a Movie by Title

Endpoint: GET /movies/:title

Description: Retrieves details of a specific movie by its title.

Path Parameters:

  • title: Title of the movie

Responses:

  • 200 OK: Returns the movie object.
  • 400 Bad Request: If the movie does not exist.

Get Movies by Genre

Endpoint: GET /movies/genre/:genreName

Description: Retrieves movies of a specific genre.

Path Parameters:

  • genreName: Name of the genre

Responses:

  • 200 OK: Returns the genre object of the first movie found with the specified genre.
  • 400 Bad Request: If the genre does not exist.

Get Movies by Director

Endpoint: GET /movies/director/:directorName

Description: Retrieves movies directed by a specific director.

Path Parameters:

  • directorName: Name of the director

Responses:

  • 200 OK: Returns the director object of the first movie found with the specified director.
  • 400 Bad Request: If the director does not exist.
